# CBT Exam System

A Computer-Based Testing system built with Node.js, Express, MongoDB, and vanilla JavaScript.

## Features

- User registration and login
- Exam selection
- Timed exams with auto-submit
- MCQ questions
- Scoring and results
- Admin panel for managing questions and exams

## Setup

1. Install MongoDB and start it.
2. cd backend
3. npm install
4. Create .env file with MONGODB_URI and JWT_SECRET
5. npm run dev
6. Open http://localhost:5001 in browser

## Usage

- Register as student or admin
- Login
- Admin: Add questions and exams
- Student: Select and take exams, view results
